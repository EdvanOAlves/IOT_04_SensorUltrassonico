# IOT_04_SensorUltrassonico
Atividade desenvolvida com ESP32 como prática do componente, usando do sensor ultrassonico como um detector de proximidade. Desenvolvido em aula da de IOT do curso Desenvolvimento de Sistemas do Senai

Segue um vídeo do código sendo executado em circuito:
https://www.youtube.com/watch?v=lQjbObJetFI

## Tecnologias utilizadas
* C++
* Arduino IDE
* Wokwi
* ESP32
